This is the solution to senate bus problem.

To run the program just run the Main.java file. Them the simulation will run in the console.

Number of riders are taken from a Random
Number of Busses needed is Number of riders / 50

Riders will be creating in a bulk of 60s

Riders trying to get to the waiting area.

Maximum of 50 riders can get in to the waiting area

All the waiting riders will enter the bus.

Riders come late cannot enter the waiting area or Board the bus. They have to wait for the next bus

Then All the busses will arrive

After all the busses departed process will terminate

There may be some waiting riders left out who cannot get in to a bus.
